WHAT EACH PART DOES
audio/

Only handles:

microphone
volume
silence detection
audio cleanup
device selection

Nothing else.

asr/

Only handles:

SimulStreaming
Whisper
Sherpa later
socket communication
transcript parsing

Nothing else.

This lets you replace:

SimulStreaming → Sherpa

without destroying the app.

Huge.

translation/

Only handles:

text translation
language selection
language rules

Nothing else.

vision/

Only handles:

camera
face detection
person tracking
OCR
object recognition

Nothing else.

subtitles/

Only handles:

subtitle smoothing
placement
HUD logic
caption flow

This becomes one of the hardest modules later.

config.py

This is VERY important.

Put things like:

SOURCE_LANGUAGE = "en"
TARGET_LANGUAGE = "bg"
DEVICE_INDEX = 25
SERVER_PORT = 43001
MODEL_SIZE = "small"

here.

Never hardcode them everywhere.

Huge best practice.
